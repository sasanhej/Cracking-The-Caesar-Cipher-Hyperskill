Description

Throughout history, people enjoyed sending secret messages. A message that only a few people can read ensures privacy, even when enemies are everywhere. Your friend has taken up ciphering as a hobby, so you will need to learn about them too. He has no good reason for this. He thinks they're neat!

In this first stage, you will convert a list of numbers into a message. Each number points to a letter in the lowercase English alphabet, and the order is simple:

\[0 = a, 1 = b, ... 24 = y, 25 = z\]

He tells you that he likes to sing the Alphabet song and count on his fingers to find the correct number. Fortunately, Python has a module string that includes string.ascii_lowercase. Searching for a string is faster than singing this song.

Objectives

In this stage, your program should:

  1. Take an input: A string of integers separated by spaces ' '
  2. Use string.ascii_lowercase to decipher this message
  3. Print the deciphered message as a string.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

> 0 1 2
abc

Example 2:

> 7 4 11 11 14
hello

Example 3:

> 2 7 4 4 18 4 2 0 10 4
cheesecake
