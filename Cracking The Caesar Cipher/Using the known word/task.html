Description

Your friend decided it is not fun to shift everything by 3 or pass along a shift number. The simple integer makes the encoded word that follows seem less fancy.

He will no longer be providing you with a shift number. Instead, he will give you a pair of messages:

  1. A sentence with the word butterscotch somewhere in it, with a secret shift value X.
  2. Another sentence (your target) with the same shift value X.

Also, he can now write encoded messages so quickly that he has also decided to send more than one word at a time.

As there is no space (" ") in this alphabet, x will be used instead. hello world would be written as helloxworld before your friend applies his shift cipher.

After deciphering, please replace any x with " ".
Objectives

In this stage, your program should:

  1. Take an input: A string of characters separated by spaces ' '
  2. Identify the shift number by finding the secret word butterscotch
  3. Take the second input: an encoded string of characters separated by spaces ' '
  4. Use the shift number to decipher the second message
  5. Replace any x letters in the deciphered message with " " space
  6. Print the second deciphered message.
Examples

The greater-than symbol followed by a space (> ) represents the user input. Note that it's not part of the input.

Example 1:

> r r r
