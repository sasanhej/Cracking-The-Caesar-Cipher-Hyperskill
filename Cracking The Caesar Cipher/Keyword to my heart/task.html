Description

Things are heating up. Something special is in the works. The messages are becoming more critical.

To handle this increasingly sensitive information, your friend has decided to upgrade his paper strips. He has agreed to produce a Vigenère cipher using handmade tabula recta (or Vigenère square).

A Vigenère square. (source)

A Vigenère square is similar to the two strips of paper he used for the Caesar cipher, but it is made of a bunch of horizontal rows, not just two. Each row shifts one step further to the left, represented by a letter.

A repeating keyword is then used to encode letters by drawing a line straight down from that letter in the top row to the row in the keyword, like in a game of Battleship. This is easier to explain with examples:

encode(message: "abcd", keyword: "AAAA") -> "abcd". Every letter in the keyword is in the top row A.

encode(message: "abcd", keyword: "BBBB") -> "bcde". Every letter in the keyword is in the next row down, B.

encode(message: "abcd", keyword: "AABB") -> "abde". The first two letters are enciphered with the A row. The second two are with the B row.

encode(message: "abcd", keyword: "AB") -> "acce". The keyword AB repeats until the message ends, becoming ABAB.

encode(message:"abcd", keyword: "CAT") -> "cbvf". As mentioned above, the keyword CAT is repeated and becomes CATC.

Thankfully, your friend is giving you some hints to help you. He gives you a regular message, the length of the keyword, and the same message encoded. The real power, he thinks, will be that he can write a message using just his Vigenère square, and you can automate a keyword-finding process to unlock his secret messages much faster than anyone who has a pen and paper.

Objectives

In this stage, your program should decode the Vigenère cipher text. This one is similar to the Caesar cipher but uses a repeating keyword to control the shifts.:

  1. Take an input: An integer representing the length of the keyword
  2. Take an input: A string of characters separated by spaces ' '
  3. Take an input: That exact string of characters separated by spaces ' ', but encoded
  4. Take an input: Your target message is encoded. A string of characters separated by spaces ' '
  5. Use the keyword to decode your target message
  6. Replace any x letters in the deciphered message with " " space
  7. Print the deciphered message.
